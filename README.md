# Event_Mangement-EaseEve 

I built EaseEve to make organizing events less of a headache. Instead of juggling spreadsheets or sticky notes, this gives you a simple web app where you can create, manage, and keep track of events all in one place.

Why I built it: I wanted a project where I could really get comfortable with core JavaScript — no frameworks to lean on, just HTML, CSS, and vanilla JS doing the actual work. Event management felt like a good real-world problem to solve while doing that.

What it does:

Lets you create and manage events with relevant details
Saves everything using the browser's Local Storage, so your data stays even after refreshing
Clean, responsive interface built from scratch with HTML5 and CSS3

Tech stack: HTML5, CSS3, JavaScript (ES6), Local Storage API

What I learned: Working without a framework forced me to actually understand how the DOM works, how to manage state manually, and how to structure a project cleanly without relying on a library to do it for me.
